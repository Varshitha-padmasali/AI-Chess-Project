# AI Chess Move Predictor

An AI-powered chess platform using Stockfish and Machine Learning.

## Features

- Predict top 3 best next moves
- Explain why each move is strong
- Live move analyzer
- Winning probability bar
- Two-player mode with AI analysis
- ML-based move quality detection

## Tech Stack

- Python
- Flask
- Stockfish
- React (planned)
- scikit-learn
- python-chess

## Current Status

Backend setup completed. Engine integration in progress.

## Future Scope

- Human move prediction
- Opening detector
- Chess insights dashboard
